# TransferWise-HM
# To start:
  - Install json-server
  - npm install -g json-server
  - go inside of the folder of the project ("transferwise-homework") and start json-server with
  - json-server --watch src/data/contacts.json


 - npm install
 - npm start

# To test:
npm test
